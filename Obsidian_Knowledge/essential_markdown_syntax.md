
# Essential Markdown Syntax


## 1. Links
You can create hyperlinks using the `[link text](URL)` syntax.

[OpenAI](https://www.openai.com)

- the `!` for display the content of the link.

## 3. Tables
You can create tables using pipes \`|\` and dashes \`-\`.

```
| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
```

| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
